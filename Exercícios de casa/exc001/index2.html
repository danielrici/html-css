Textos excluídos

Os textos excluídos podem ser usados para indicar que esse trecho não é mais importante no texto. Você pode excluir um texto usando a tag <del>.

Listas

Listas ordenadas

As listas ordenadas são listas que têm uma ordem. Elas podem ser ordenadas com números, letras ou números romanos em HTML5. Para fazer uma lista ordenada, você usa a tag <ol> e uma tag <li> para cada elemento da lista.

Listas não ordenadas

As listas não ordenadas são listas que não tem uma ordem. Em HTML5, você pode marcar listas não ordenadas com círculos, discos, e quadrados. Para fazer uma lista ordenada, use a tag <ul> para fazer a lista e para cada elemento da lista também use a tag <li>.

Sobrescritos e Subscritos

Sobrescritos

Os subrescritos podem aparecer em algum lugar como x3+5. Ele pode ser representado com a tag <sup>.

Subcritos

Os subscritos podem ser representados com a tag <sub> e podem aparecer em lugares como H2O.

Texto invertido

Para fazer texto invertido, basta usar a tag <bdo>. Você pode escolher os parâmetros ltr e rtl, mas para fazer o texto invertido, você precisa do parâmetro rtl.
Exemplo:
ltr: Olá, tudo bem?
rtl: Olá, tudo bem?
