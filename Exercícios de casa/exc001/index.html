Resumo HTML5

Hirearquia de títulos

Você pode usar a tag <h1> para criar títulos de primeira ordem. Para títulos "menores", como Subtítulos, você pode usar a tag de h e depois a ordem daquele título. Por exemplo, a tag <h4> é usada para fazer um título de 4ª ordem.

Textos em negrito

Antigamente usavamos a tag B para deixar os textos deste jeito, mas agora é mais recomendavel usar a tag STRONG porque eventualmente ela pode se tornar obsoleta.

Exemplos de textos em negrito:

Tag B: Lorem, ipsum dolor sit amet consectetur adipisicing elit.

Tag STRONG: Lorem ipsum dolor sit amet consectetur adipisicing porro obcaecati totam quos.

Como você viu, não há diferença visual nos textos.

Textos em Ênfase / Itálico

O mesmo se vai com os textos em itálico. Antes usavamos a tag I para coloca-los em ênfase, mas hoje é mais recomendável você usar a tag EM para isso, mesmo que não tenha uma diferença visual.

Exemplos de textos em itálico/ênfase:

Tag I: Lorem ipsum dolor sint cum fuga impedit, cumque nisi, officiis cupiditate.

Tag EM: Lorem quam aut dele molestiae reprehenderit? Laboriosam.

Textos sublinhados

Você pode criar um texto sublinhado usando a tag U, que não é semântica, mas mesmo assim muito usada.

Exemplos de sublinho:
Lorem ipsum dolor sit aipisicing elit. Pariolsi!
Lorem ipsum dolor sit amet consectetur ada ae.
